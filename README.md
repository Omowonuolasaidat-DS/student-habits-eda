Student Habits Analysis

Project Overview

This project analyzes student study habits and academic performance using Python, Pandas, and Matplotlib.

The dataset includes:

- Student names
- Study hours
- Sleep hours
- Attendance
- Scores

The project performs:

- Data analysis
- Filtering
- Creation of new columns
- Performance classification
- Data visualization

Features

- Finds students with the highest attendance
- Calculates average score
- Filters students studying above average hours
- Creates a Performance_Status column
- Generates:
  - Bar chart of scores
  - Line chart of study hours
  - Pie chart of performance status

Technologies Used

- Python
- Pandas
- Matplotlib
